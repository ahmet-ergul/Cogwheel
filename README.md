# Cogwheel

A node-based visual factory planner for Minecraft 1.21.1 NeoForge — think "n8n for modded Minecraft." Plan your Create-mod factories (and any other recipe-driven mod) as a graph on a canvas before you build them in the world.

## Requirements

- Minecraft `1.21.1`
- NeoForge `21.1.228` or newer (in the `21.1.x` line)
- Java 21 (bundled with the modern Minecraft launcher)

## Development

```sh
./gradlew runClient   # launches a dev MC client with the mod loaded
./gradlew build       # builds the mod jar
```

The default keybind is **`K`** (rebindable under `Options → Controls → Cogwheel`).
